Contains classes and interfaces that are useful for dealing with {@link java.lang.Throwable}s.

See {@link com.edugility.throwables.ThrowableList} for a class that assists with reading a {@link java.lang.Throwable}'s {@linkplain java.lang.Throwable#getCause() causal chain}.

See {@link com.edugility.throwables.ThrowableChain} for a class that helps with capturing all {@link java.lang.Throwable}s thrown, even in the case of nested {@code try}/{@code catch}/{@code finally} blocks.
